This is the source code for the game Mecha Walker, which was made for the Ludum Dare 54 game jam. The game is playable via browser here
https://le-don.itch.io/mecha-walker

The game was made in Godot.

The idea behind Mecha Walker is building a robot with some parts and travel from the start point to the end, while visiting various towns and accepting jobs to earn money to buy new parts and ressources.

Inspirations behind the game were titles like Front Mission and Oregon Trail.
